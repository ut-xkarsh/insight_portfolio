Ambulance Booking System

A real-time desktop application to streamline ambulance requests and dispatch for emergency services using Python Tkinter and MySQL.

The Challenge & My Approach

Existing emergency services often face delays in ambulance dispatch. My goal was to create a **rapid, efficient, and reliable booking system** to connect patients with available ambulances, focusing on **real-time location tracking and optimized routing** within a desktop environment.

My Contribution:

  • Developed the **Python backend** for managing patient requests and driver-side updates, interacting with MySQL.
  • Implemented **user authentication** via a secure **Tkinter login interface**.
  • Designed and built the core logic for **dispatch and status updates**.
  • Managed the **MySQL database** for storing user, driver, and request data.

Technologies Used:

Python Tkinter MySQL Desktop Application Database Management

Outcome & Impact:

This system automates the matching process, potentially **reducing ambulance dispatch time and improving response efficiency** in critical situations.
